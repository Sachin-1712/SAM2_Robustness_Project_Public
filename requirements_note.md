# Dependencies

Approximate list of dependencies required for this project:

- **Python**
- **Conda Environment Name:** `sam2`
- `torch`
- `torchvision`
- `opencv-python`
- `numpy`
- `pandas`
- `matplotlib`
- `tqdm`
- `albumentations` (only required for the legacy `apply_weather.py` script)
- **SAM2 package:** Required for inference
